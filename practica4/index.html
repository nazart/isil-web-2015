Practica 4

Para esta practica se debe usar el siguiente web service
http://www.webservicex.net/periodictable.asmx
El WSDL para usar es: "http://www.webservicex.net/periodictable.asmx?WSDL"
Reglas:
  • Se debe de obtener todos los elementos registrados en el servicio GetAtoms y guardarlos en una tabla (8 Puntos)
  • Se debe de consumir el servicio "GetAtomicNumber" para obtener el detalle de cada elemento (6 puntos)
  • Se debe de exportar a excel y pdf la lista de la tala periodica (6 puntos)
